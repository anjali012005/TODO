# TODO App

This is a simple and interactive TODO app that helps you manage your daily tasks efficiently.  
You can easily **add new tasks**, **view them**, and **delete tasks** once completed.  

<img width="1902" height="962" alt="image" src="https://github.com/user-attachments/assets/aa0c276d-5932-4f79-adff-3f8518e30bde" />


## Features
- Add tasks with a simple input field.
- Delete tasks that are no longer needed.
- Clean and user-friendly interface.

This app is perfect for keeping track of your to-dos and staying organized.
